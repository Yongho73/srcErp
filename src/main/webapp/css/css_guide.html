    위의 이미지 예제와 같이 a태그를 button태그로 변경하고 onclick에 return false를 추가하여 사용합니다.

   - onfocus="this.blur()" 사용하면 키보드 접근불가 이므로 접근성 부적합

   - window.open()을 통해 새창 띄울시, title,alt 속성에 새창이라고 명시해줘야하하며
파라메터가 _blank, _self, _parent, _top중 하나를 포함하고 있으면 적합

   

 


<iframe>태그의 경우 title 속성 반드시 입력!!

 

예) <iframe title="레이아웃 프레임">

 


 

<input>태그의 경우 label 태그와 for속성 반드시 포함!! (type=text,radio,checkbox,file,submit 동일) 

예) 이름 : <input id="name" type="text"> (X) 

           <label for="name">이름 :</label><input type="text" id="name"> (O) -  (label의 for속성과 id를 같게 해주어 연결)

 

    ※ label이 없는 input의 경우

      <input id="zipcode2" type="text" title="우편번호 뒤 세자리">

 

<select>태그의 경우 title속성과 label 태그 + for속성 반드시 포함!!

예) 직업:<select id="job">

               <option value="student">학생</option> (X)

 

   <label for="job">직업:</label><select id="job"> -  (label의 for속성과 id를 같게 해주어 연결)

               <option value="student">학생</option>(O)

 

    ※ label이 없는 select 디자인의 경우 title로 대체

      <select id="job" title="직업 선택"> 

 

 

 

<textarea>태그의 경우 title 속성과 label 태그 반드시 포함!!

 

예) 내용: <textarea title="내용을 입력합니다.">


<html>태그의 경우 lang 속성(기본 언어 설정) & title 태그 반드시 포함!! 

예) <html lang="ko"> 

        <title>X-ERP</title>

 

 

<img>태그의 경우 alt 속성 반드시 입력!!

 

예) <img src="logo.png" alt="로고이미지">
